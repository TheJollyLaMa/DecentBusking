## Overview

Create `decent.config.js` as the global configuration script for the DecentBusking app. This file should be loaded before any other modules (via a `<script>` tag in `index.html` before `type="module"` scripts).

## Purpose

Serves as the **single source of truth** for all app-wide constants — chain info, contract addresses, IPFS settings, and branding.

## Implementation

The file should export (or assign to) `window.DecentConfig` with the following fields:

```js
window.DecentConfig = {
  appName: "DecentBusking",
  chainId: 137,                        // Polygon mainnet (or 80001 for Mumbai testnet)
  contractAddress: "0x...",           // DecentNFT contract address
  w3upSpaceDID: "did:key:...",        // w3up space DID for IPFS uploads
  ipfsGateway: "https://w3s.link/ipfs/",
  tokenSymbol: "MATIC",
};
```

## Acceptance Criteria

- [ ] `decent.config.js` exists at the repo root (or `/js/decent.config.js`)
- [ ] `window.DecentConfig` is set with all 6 fields above
- [ ] File is loaded via `<script src="...decent.config.js"></script>` in `index.html` **before** any `type="module"` scripts
- [ ] A comment block at the top explains each field
- [ ] Placeholder values are clearly marked with `// TODO: replace` comments

## Notes

- All other modules (`stage.js`, `mint.js`, `space.js`, etc.) should read from `window.DecentConfig` rather than hardcoding values.
- Keep this file plain JS (no ES module syntax) so it loads synchronously and is available globally.
